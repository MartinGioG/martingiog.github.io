Objetivo del Proyecto

Desarrollar un sistema de análisis integral para plataforma de e-commerce que permita monitorizar KPIs críticos, entender el comportamiento del usuario y optimizar la conversión.

Arquitectura de la Solución

Implementé una arquitectura completa de Business Intelligence:

  • Fuentes de Datos: Google Analytics, Base de datos MySQL, API de Shopify
  • ETL: Power Query para transformación y limpieza de datos
  • Modelo de Datos: Estrella con tablas de hechos y dimensiones
  • Visualización: Dashboards interactivos con DAX avanzado

Métricas Principales

  • Conversión: Tasa de conversión por canal y dispositivo
  • Funnel: Análisis de embudo de conversión
  • Comportamiento: Tiempo en sitio y páginas por sesión
  • Ventas: Revenue, AOV y análisis de cohortes

Impacto del Proyecto

  • Incremento del 25% en tasa de conversión
  • Identificación de cuellos de botella en el funnel
  • Optimización del 30% en presupuesto de marketing
  • Reducción del 70% en tiempo de análisis

Stack Tecnológico

Power BI SQL Power Query DAX Google Analytics

Características

  • Diseño mobile-first
