Objetivo del Dashboard

Este dashboard fue desarrollado como herramienta de análisis para el proyecto Procaflor, permitiendo el monitoreo de indicadores financieros clave y la toma de decisiones basada en datos para la implementación de servicios IoT en el sector cafetalero.

Metodología y Técnicas Excel Aplicadas

Para el análisis del proyecto Procaflor, utilicé las siguientes técnicas avanzadas de Excel:

  • Tablas Dinámicas: Análisis multidimensional de costos por tipo de productor (pequeño/mediano)
  • Gráficos Dinámicos: Visualización de flujos de caja proyectados (años 1-5)
  • Fórmulas Avanzadas: Cálculo de VAN, TIR, ROI y punto de equilibrio financiero
  • Segmentación de Datos: Filtros interactivos por departamento (Ahuachapán/Sonsonate)
  • Modelos Financieros: Proyecciones de crecimiento de clientes (87 a 348 productores)
  • Análisis de Sensibilidad: Escenarios con variaciones en precios y costos

Insights y Hallazgos

El análisis en Excel permitió identificar oportunidades clave para el proyecto Procaflor:

  • Los costos de administración representan el 58.23% del presupuesto total
  • El crecimiento proyectado muestra viabilidad a partir del año 3 con 209 productores
  • La relación Beneficio/Costo de 0.98 es aceptable considerando impactos sociales
  • El análisis de sensibilidad revela vulnerabilidad a baja adopción de clientes

Sobre Procaflor
