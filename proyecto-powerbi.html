Objetivo del Proyecto

Desarrollar un sistema de análisis integral que permita a la dirección comercial tomar decisiones basadas en datos sobre estrategias de ventas, distribución de recursos y optimización de inventario.

Metodología

Utilicé un proceso ETL (Extract, Transform, Load) completo:

  • Extracción: Consolidación de datos desde múltiples fuentes (CRM, ERP, archivos CSV)
  • Transformación: Limpieza y estandarización de datos usando Power Query
  • Carga: Alimentación del modelo de datos para el análisis
  • Visualización: Creación de dashboards interactivos con tablas dinámicas y gráficos

Resultados Alcanzados

  • Identificación de patrones de venta estacionales
  • Optimización de inventario con reducción del 15% en costos
  • Segmentación efectiva de clientes por valor y potencial
  • Reducción del 60% en tiempo de generación de reportes
